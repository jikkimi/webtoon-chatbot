# Webtoon Deep Dive

A premium webtoon exploration service with an AI-powered recommendation chatbot.

## Features
- **AI Chatbot**: Personalized recommendations using Gemini 1.5 Flash.
- **Rich UI**: Modern dark theme with responsive grid and smooth animations.
- **Search & Filter**: Find webtoons by title, author, genre, or age rating.
- **Detail View**: View full story details and explore similar works.

## Deployment
This project is a standalone HTML/JS application. It can be deployed directly to Vercel or GitHub Pages by selecting the root directory.
