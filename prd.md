# 웹툰 탐색 & 챗봇 추천 서비스 기획안 (비로그인 MVP)
# 서비스명: Webtoon Deep Dive


## 1. 서비스 개요


### 목적
- 보유한 웹툰 DB(JSON 기반)를 활용하여
 - 리스트/검색/필터 중심의 웹툰 탐색 제공
 - 챗봇을 통한 취향 기반 웹툰 추천 제공
- 로그인 없이 즉시 사용 가능한 구조


### 운영 원칙
- 회원가입/로그인 없음
- 관심 등록/찜/히스토리 저장 없음
- 모든 사용자 상태는 세션 내에서만 유지 (새로고침 시 초기화)


---


## 2. 주요 기능


## 2.1 웹툰 리스트 디스플레이


### 1) 카드형 리스트 UI
- 썸네일
- 제목
- 작가
- 장르
- 태그
- 연령등급 배지


### 2) 정렬 기능
- 제목순
- 추천순(챗봇 매칭 결과 반영 시)


### 3) 필터 기능
- 장르 필터 (판타지, 로맨스, 스릴러 등)
- 연령등급 필터 (전체/15/19/정보없음)
- 태그 기반 필터 (예: 회귀, 학원, 오피스, 헌터물 등)


### 4) 검색 기능 (로컬 검색)
- 제목
- 작가
- 태그
- 줄거리 키워드


실시간 키워드 매칭 기반 필터링


---


## 2.2 웹툰 상세 페이지


### 제공 정보
- 썸네일
- 제목 / 작가
- 장르 / 태그
- 연령등급
- 줄거리


### 액션
- "바로 보기" (원문 링크 이동)
- "비슷한 작품 보기"
 - 동일 장르/태그 기반 자동 추천 5개 노출
 - 별도 저장 없이 즉시 제안


---


## 3. 챗봇 기능 설계


## 3.1 목적
사용자의 자연어 입력을 기반으로
웹툰 DB 내 작품을 검색/매칭하여 추천 제공


---


## 3.2 대화 흐름 (MVP)


### Step 1. 취향 수집
- "어떤 장르 좋아하세요?"
- "원하는 분위기 있어요? (가벼운/어두운/힐링/피폐 등)"
- "특정 키워드 있을까요? (회귀, 학원, 오피스, 헌터물 등)"
- "19금은 제외할까요?"


### Step 2. 조건 정리
- 장르
- 분위기
- 키워드
- 연령 제한 여부


### Step 3. 후보 추천
- 5~10개 카드 형태로 제시
- 각 작품마다:
 - 간단 추천 이유 1~2문장
 - "바로 보기" 버튼


### Step 4. 재추천 루프
사용자 자연어 피드백 기반 재매칭:
- "좀 더 어둡게"
- "로맨스 빼고"
- "액션 더 강한 걸로"
- "비슷한데 더 가벼운 걸로"


※ 모든 선호도는 세션 내에서만 반영 (저장하지 않음)


---


## 4. Gemini 연동 설계 (gemini-2.5-flash-lite)


### 역할 1: 웹툰 검색 & 매칭
입력:
- 사용자 자연어 취향 문장
- 웹툰 DB 데이터 (title, story, genre, tag 등)


출력:
- 매칭 점수 기반 후보 정렬
- 상위 5~10개 추천 리스트


---


### 역할 2: 추천 사유 생성
각 작품에 대해:
- "이 작품이 맞는 이유" 1~2문장 생성
- 장르/태그/줄거리 기반 설명


예:
> 회귀 키워드와 성장형 판타지 요소가 강해서 요청하신 분위기와 잘 맞습니다.


---


### 역할 3: 분위기 샘플 대사 생성 (선택 기능)


사용자가 요청 시:
- 원작 대사 재현이 아니라
- 줄거리/태그 기반 "스타일 창작 대사" 생성


예:
> “이번 생은 다를 거야. 내가 약자였던 시간은 여기서 끝내.”


※ 저작권 침해 방지를 위해 실제 원작 대사 재현 금지


---


## 5. 데이터 처리 설계


### 1) 장르/태그 표준화
- 문자열 혼합 구조 정리
- 장르 배열과 태그 배열 분리
- 중복/개행 제거


### 2) 연령등급 처리
- Unknown 값은 "등급 정보 없음"으로 표시
- 필터에서 별도 그룹 제공


---


## 6. 상태 관리 정책


### 유지
- 현재 대화 맥락 (세션 내)


### 제거
- 관심 등록
- 찜 목록
- 사용자 프로필
- 히스토리 저장
- 개인화 학습


---


## 7. 화면 구성 (MVP 범위)


### 1) 홈 (탐색)
- 검색바
- 필터칩
- 카드 리스트


### 2) 상세 페이지
- 작품 정보
- 바로 보기
- 비슷한 작품 제안


### 3) 챗봇 화면
- 하단 입력창
- 상단 추천 카드 노출 영역
- 재추천 인터랙션 지원


---


## 8. MVP 범위 요약


- 로그인 없음
- 저장 기능 없음
- 리스트/검색/필터/상세/바로 보기 제공
- 챗봇 기반 취향 추천
- Gemini 기반:
 - 검색/매칭
 - 추천 이유 생성
 - 분위기 샘플 대사 생성 (옵션)
